Video Game Sales API

This API provides information about Video Game Sales.

Get All Films

Retrieve a list of all Video Game Sales.

GET https://video-game-sales-api.vercel.app/api/allsales/

Get Platform Sales Data

(eg. Xbox, Playstation)

GET https://video-game-sales-api.vercel.app/api/platform/{platform name}

Example: https://video-game-sales-api.vercel.app/api/platform/xbox
